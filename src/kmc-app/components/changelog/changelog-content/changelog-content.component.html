KMCng v4.4.0 (2018-04-30)

New Functionality

Platform

  • De-Flash all KMC and its components.
  • A new engaging design to give a unified, consistent user experience throughout the platform. All pages were revised and improved.
  • Performance improvement.

Content creation and editing

  • Support uploading videos greater than 2GB.
  • Upload Monitoring – presenting you all current uploads status. Located in to top navigation bar, for quick access from any page on the KMC.
  • Kaltura New Editing Tool is now available from the KMC as well – Clip, Trim, advertisement and more.
  • IVQ creation.

Entries list & details

  • YouTube entries are visible in the entries list.
  • Each entry page includes a fixed player and entry details. Those will be available also while navigation between the entries different tabs.

Share & Embed

  • Sort the players list is available now supporting sort by name, upload date, update date.
  • Updated players list - all irrelevant and old players were removed.

Deprecated Functionality

  • KMC dashboard was removed. The content page is now the KMC landing page. Account and Billing information can be found now in the Account Information page.
  • Notification setting (integration setting) via the KMC – new notification setup is done now by the new notification module.
  • Import from Web – ingest video from Metacafe video repository.

Missing Functionality

The following functionality is currently not supported, and will be deployed in the future versions –
  • Foreign languages support – currently only English is supported.
  • Analytics.
  • Replace Video.
  • Entry Flavors actions - drop folders and remote storage.
  • Thumbnail cropping.
  • Forgot password.
  • iTunes Syndications.
  • Syndication: instruction screens are currently not available for all channels.
  • Small resolutions support (1024x768).
  • Help – for the Beta, user guide is provided in PDF file.

Known Limitations

  • Table columns size currently cannot be modified.
  • When trimming a video – in case customer’s configuration requires ‘approval before replacing’, the trim activity could not be fully completed, replacement should be approved in the current KMC.
  • Navigation breadcrumbs - back arrow on entry details page links to the entries list page, and not necessarily to the latest page.
  • The Upload Monitoring page is updated every 10 seconds, changes to the uploads status will be reflected near real time.
  • Login – email format validations and error messages covers most of the invalid formats, some formats are not validated.
  • Open the same KMC in 2 tabs in the same browser, requires a login to each one of the pages.
